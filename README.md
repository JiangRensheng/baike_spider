#程序简介#

	baike_spider是一个可以爬取百度百科词条中所有相关词条的简介并输出的小程序。
    这个Baike_spider是[慕课网](http://www.imooc.com)的[Python开发简单爬虫](http://www.imooc.com/learn/563)系列课程的一个实例修改而来的。主要是把所有的代码写成了一个文件，优化了输出的HTML代码，以及添加了root_url（起始链接）和times（爬取次数）的输入交互。
